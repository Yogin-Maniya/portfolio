Transforming Urban Transit

RMTS (Rajkot Mass Transport Service) is an innovative digital platform developed using .NET for both the frontend and backend, with SQL Server as the database. The application empowers users to enter their starting and ending points to search for available bus routes. It then displays a variety of bus cards representing different routes.

By clicking on any bus card, detailed route information is revealedâ€”including distance, travel time, directional maps, and expected arrival times. The platform further enhances user experience by automatically detecting the user's current location to display nearby pickup points within a 3 km radius, complete with distance metrics.

Additionally, RMTS provides a comprehensive listing of all Rajkot routes with detailed information, and it incorporates an integrated complaint and feedback section where users can select a bus number to register issues and share feedback.

Key Features

  • Input start and end points to search for bus routes
  • Display of bus cards with route details and quick view options
  • Detailed view showing distance, travel time, map directions, and arrival times
  • Current location detection with nearby pickup points (within 3 km)
  • Comprehensive listing of all Rajkot routes with detailed information
  • Integrated complaint and feedback section for bus-specific issues

Technical Implementation

RMTS is built using .NET for both the frontend and backend, ensuring a robust and scalable architecture. SQL Server handles all data management efficiently. The system leverages real-time geolocation services and advanced mapping libraries to offer dynamic route planning, current location detection, and detailed navigation maps. Furthermore, its modular design seamlessly integrates a complaint and feedback module for enhanced user interaction.
