RMTS APP – Revolutionizing Public Transit

RMTS APP is a full‑stack public transportation management solution that leverages Flutter for its dynamic UI, .NET for robust backend services, and SQL Server for reliable data management. This application mirrors the features of RMTS WEB, offering real‑time bus schedules, route maps, and detailed transit information—all in a powerful mobile app format.

Users can search for routes by entering start and end points, view live transit data including distances and arrival times, and access additional features like feedback submission and complaint registration.

Key Features

  • Input start and end points to search for bus routes
  • Display of bus cards with route details and quick view options
  • Detailed view showing distance, travel time, map directions, and arrival times
  • Current location detection with nearby pickup points (within 3 km)
  • Comprehensive listing of all transit routes with detailed information
  • Integrated feedback and complaint module

Technical Implementation

RMTS APP is built using Flutter for the frontend, with a .NET backend ensuring scalability and robustness, and SQL Server managing the data efficiently. This full‑stack solution delivers a seamless, responsive, and secure experience, making public transportation management more efficient and user‑friendly.
